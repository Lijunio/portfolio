Projeto 1

Amigo Oculto Natal Família Souza

Este projeto utiliza JavaScript, HTML/CSS para organizar e executar o sorteio do amigo oculto entre membros da família. Oferece sorteio aleatório, restrições personalizadas e feedback interativo por meio de um pop-up. Ao final do sorteio, o sistema gera automaticamente um arquivo de resultado para download, facilitando a distribuição dos resultados entre os participantes. Eu fiz esse site pois não queria pagar sorteio em outros sites, pois tinha muitas restrições.

100% Concluído

Projeto 2

Bioguard

Integrando um leitor de biometria em um sistema de registro de ponto completo, o BioGuard utiliza JavaScript, HTML/CSS no frontend e Node.js com MySQL no backend. Oferece funcionalidades robustas, como criação e consulta de registros de ponto, gestão de funcionários e departamentos. A interface intuitiva e responsiva diferencia funcionários de gerentes no login, garantindo segurança e precisão no controle de acesso. Projetado para modernizar e automatizar processos de registro de ponto.

100% Concluído

Projeto 3

Consultoria Academica

Website de consultoria acadêmica desenvolvido utilizando as linguagens HTML, CSS, JavaScript e PHP com o intuito de oferecer uma experiência fluida e interativa aos usuários. A interação permite explorar a variedade de serviços acadêmicos ofertados. Além disso, foi incluída a funcionalidade de solicitação de orçamentos através do preenchimento de formulários e conexão direta com o e-mail e contato direto através de API com o WhatsApp do consultor para resolução de dúvidas.

40% Concluído

Bioguard

Este projeto integra um leitor de biometria em um sistema completo de registro de ponto para funcionários. Utilizando JavaScript, HTML/CSS no frontend e Node.js com MySQL no backend, o BioGuard oferece funcionalidades robustas de gerenciamento de dados. O sistema inclui a criação e consulta de registros de ponto, gestão de funcionários e departamentos, tudo exibido de forma intuitiva e responsiva. A interface foi desenvolvida para ser amigável e eficiente, garantindo a segurança e precisão no controle de acesso dos funcionários. A página de login possui opções distintas para funcionários e gerentes, cada uma redirecionando para funcionalidades específicas. Este projeto foi desenvolvido para atender às necessidades de empresas que buscam modernizar e automatizar seus processos de registro de ponto, proporcionando uma solução confiável e integrada.

100% Concluído
